# S5-T2 — Reporting views (общее описание)

**Для борди:** короткий опис задачі для швидкого ознайомлення.


**Приоритет:** Should


## Цель
Генерировать управленческие представления из канонической модели без дублирования источника истины.

## Что делаем
- Шаблоны: executive, consultant, problem map, recommendations.
- Маппинг model -> markdown views.
- Согласованность терминов с глоссарием.

## Критерий успеха
Отчеты понятны ролям пользователей и соответствуют одному каноническому состоянию кейса.
