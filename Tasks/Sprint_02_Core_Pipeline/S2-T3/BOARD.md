# S2-T3 — Canonical case model + index + versioning base (общее описание)

**Для борди:** короткий опис задачі для швидкого ознайомлення.


**Приоритет:** Should


## Цель
Собрать каноническую модель кейса и индекс быстрого доступа для работы следующих модулей.

## Что делаем
- Формируем `case_model.json` как single source of truth.
- Генерируем `model_index.json` и `model_version.json`.
- Готовим базовый механизм snapshot versioning.

## Критерий успеха
Модель кейса доступна по ID-фрагментам без загрузки полной истории.
