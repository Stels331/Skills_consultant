# S1-T2 — Схемы и валидация артефактов (общее описание)

**Статус:** DONE

**Для борди:** короткий опис задачі для швидкого ознайомлення.


**Приоритет:** Must


## Цель
Ввести строгие JSON-схемы ключевых артефактов, чтобы состояние кейса было машинно-проверяемым.

## Что делаем
- Создаем `schemas/*.schema.json` для core-объектов.
- Добавляем локальный валидатор артефактов.
- Интегрируем валидацию в workflow команд разработки.

## Критерий успеха
Любой некорректный JSON артефакт обнаруживается до перехода к следующей фазе.
