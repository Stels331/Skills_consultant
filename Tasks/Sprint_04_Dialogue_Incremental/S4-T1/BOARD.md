# S4-T1 — Dialogue manager + selective context loader (общее описание)

**Для борди:** короткий опис задачі для швидкого ознайомлення.


**Приоритет:** Must


## Цель
Сделать рабочий dialogue-mode без загрузки полной истории кейса в контекст.

## Что делаем
- Вводим `dialogue_state.json` и легковесный `conversation_log.json`.
- Загружаем только релевантные фрагменты модели.
- Отслеживаем budget в `context_load_stats.json`.

## Критерий успеха
Каждый turn укладывается в лимит контекста и обновляет только нужные части кейса.
