# S3-T2 — Problem localization + Problem factory (общее описание)

**Для борди:** короткий опис задачі для швидкого ознайомлення.


**Приоритет:** Must


## Цель
Перейти от симптомов к управляемому набору проблем и выбрать Goldilocks problem для цикла.

## Что делаем
- Разделяем symptoms, likely causes, root causes, confounders.
- Формируем problem archive и текущий portfolio.
- Генерируем comparison/acceptance spec.

## Критерий успеха
Проблема сформулирована так, чтобы по ней можно было сравнивать и выбирать варианты решений.
