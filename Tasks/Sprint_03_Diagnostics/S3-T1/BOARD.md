# S3-T1 — Epistemic analysis (общее описание)

**Для борди:** короткий опис задачі для швидкого ознайомлення.


**Приоритет:** Must


## Цель
Ввести слой epistemic-контроля: качество evidence, доверие, карта неопределенностей.

## Что делаем
- Статусы claims и trust F/G/R.
- Качество evidence: источник, свежесть, валидность scope.
- Формируем `epistemic_map.json` и `evidence_quality_report.json`.

## Критерий успеха
Система отделяет доказанные утверждения от допущений и не разрешает решения без основания.
