# Solution Space Meta-Model

## Weak Intervention Class
- principle: Класс weak-interventions описывает минимально достаточные и обратимые меры, которые меняют правила входа, фильтрацию, маршрутизацию или локальные роли без тяжелой перестройки архитектуры.
- purpose: represent a reusable intervention class, not a case-specific patch
- selection_rule: keep this class in the portfolio only if it adds a distinct pareto profile or a safer rollout path
- instances: sol_00_status_quo, sol_01_commodity_pivot

## Medium Intervention Class
- principle: Класс medium-interventions описывает меры, которые частично отчуждают повторяемую экспертную функцию в метод, инструмент или отдельный операционный контур, но не требуют полной трансформации системы.
- purpose: represent a reusable intervention class, not a case-specific patch
- selection_rule: keep this class in the portfolio only if it adds a distinct pareto profile or a safer rollout path
- instances: sol_02_offtake_monetization

## Strong Intervention Class
- principle: Класс strong-interventions описывает меры, которые меняют саму топологию системы, архитектурные границы, оргструктуру или бизнес-модель, когда локальные меры уже недостаточны.
- purpose: represent a reusable intervention class, not a case-specific patch
- selection_rule: keep this class in the portfolio only if it adds a distinct pareto profile or a safer rollout path
- instances: sol_03_thermal_capex_refactoring

## sol_00_status_quo
- type: process
- assurance_level: high
- intervention_force: weak
- relevance_basis: pareto_relevant
- предприятие_продолжает_работу_в_текущем_режиме._распиловка_генерирует_700: 900 м³ отходов в месяц. Вакуумные сушилки работают от коммерческой электросети. Вывоз отходов осуществляется ситуативно, по мере переполнения бункеров, за счет предприятия (статья расходов).
- *физический_коллапс: Остановка лесопилки каждые 3-5 дней из-за переполнения бункеров (`CHR-02-WASTE-ACCUM` > 0).

Оплата коммерческой электросети для 30-дневного цикла сушки (`CHR-01-ENERGY-OPEX`) в сумме с затратами на утилизацию отходов делает себестоимость 1 м³ термодерева выше его рыночной цены. Cash Burn Rate приведет к исчерпанию ликвидности в течение нескольких недель.

## sol_02_offtake_monetization
- type: process
- assurance_level: low
- intervention_force: medium
- relevance_basis: rollout_relevant
- *организация: Введение новой роли — Waste Manager. Его единственный KPI — обеспечение `CHR-02-WASTE-ACCUM = 0` через заключение жестких off-take контрактов с производителями пеллет, МДФ или агрокомплексами.
- *бизнес: модель:* Отходы переводятся из статуса "мусор" в статус "побочный продукт". Выручка от продажи 700-900 м³ опилок/щепы целевым образом направляется на субсидирование счетов за коммерческую электроэнергию для вакуумных прессов.
- *ит: Внедрение системы диспетчеризации транспорта (TMS) для синхронизации скорости распиловки со скоростью подачи фур-щеповозов. Если фуры нет — пила автоматически блокируется (АСУ ТП интеграция).
- узел_неопределенности: Существует ли в радиусе рентабельной логистики покупатель, готовый стабильно забирать 900 м³ отходов и платить за них цену, достаточную для компенсации энерготарифов до уровня маржи >15%? (Explore рынка отходов).
- сложная_транспортная_логистика,_gr/b2b: переговоры с крупными переработчиками древесины, диспетчеризация.
- низкий_assurance_level._высока_вероятность,_что_рыночная_стоимость_сырых_опилок_не_покроет_колоссальные_затраты_на_30: дневное питание вакуумных прессов от сети. Риск нарушения Анти-Гудхарт (соблазн сжечь отходы в поле, если фура не приехала).

## sol_03_thermal_capex_refactoring
- type: architecture
- assurance_level: high
- intervention_force: strong
- relevance_basis: rollout_relevant
- радикальное_физическое_решение_проблемы_путем_возврата_к_изначальной_архитектурной_парадигме_"отходы: Энергия", но с заменой изъятого компонента (газогенератора) на доступный аналог.
- *архитектура/capex: Закупка и монтаж промышленных твердотопливных котлов (с автоматической шнековой подачей) и теплообменников для вакуумных прессов.
- *процесс: 100% генерируемых отходов (700-900 м³) направляется в топки котлов. Это гарантированно обеспечивает `CHR-02-WASTE-ACCUM = 0` (легальная внутренняя утилизация).
- *экономика: Котлы генерируют дешевую тепловую энергию для 30-дневного цикла сушки. Потребление дорогой сетевой электроэнергии снижается только до уровня питания насосов и автоматики, что радикально роняет `CHR-01-ENERGY-OPEX` и возвращает маржинальность >15%.
- *ит: Внедрение телеметрии котлов и интеграция с АСУ ТП сушилок для жесткого контроля температурных режимов (защита качества `CHR-04-QUALITY`).
- узел_неопределенности: Технология понятна (Exploit). Неопределенность лежит только в сроках поставки оборудования и наличии достаточного капитала (CAPEX) для прохождения "Долины смерти" до запуска котлов.
- кассовый_разрыв_в_период_закупки_и_монтажа_(может_занять_1: 3 месяца). Ошибки в расчете теплотворной способности сырых опилок (может потребоваться предварительная подсушка топлива).
